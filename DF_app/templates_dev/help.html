Tree Inventory
Species, distribution, time period

The Tree Inventory Data page allows you to see the distribution of one or more types of trees in Maine in a specific time period.

Example

View the distribution of pine trees with diameter at breast height between 20 and 25 cm.

Open Tree Inventory →
Environmental Data
Temp, humidity, climate

The Environmental Data page allows you to view a specified region in Maine colored by whether it fits a set of environmental criteria.

Example

See which locations in Kennebec County have more or less than 170 frost-free days a year.

Open Environmental Data →
Tree Environmental Preferences
Feasibility maps, species suitability

The Tree Environmental Preferences page allows you to create a feasibility map for a specific tree species based on selected properties.

Example

See where the elevation is and isn't suited to American Beech trees.

Open Tree Environmental Preferences →
